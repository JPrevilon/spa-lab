# SPA Portfolio Platform Lab

A single-page React application that allows users to view, add, and search personal projects dynamically.

## Features
- Displays a list of projects
- Add new projects using a form
- Search projects dynamically
- Responsive design inspired by a provided mock-up

## Technologies Used
- React
- JavaScript
- CSS
- Create React App

## Installation & Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000 in your browser

## Notes
This project uses local React state only. No backend or database is included.
